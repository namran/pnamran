pnamran by namran

simplify web app to store data into mysql but use combination of php and .htaccess to make it appear like static content

.. this is meant to save some space by dumping all html code.. image file into mysql database instead of real file.. as on some hosting.. the mysql database usages somehow unlimited .. while the real file got limited quota. I am doing this because.. my file space at my current hosting plan .. is almost full.. reaching above 90% limit.. hence my blog update have been slowed down.. too reluctant to upload more photos / screenshot...

Dependencies

imagemagick , a httpd webserver which can understand .htaccess , php interpreter perhaps.. YOU!!

Install

to use : 1. just import pnamran.sql into your mysql database. 2. edit the docroot/inc/db_config.php to reflect your database environment. 3. upload/put the whole file inside docroot into your webserver directory . 4. to create page. use phpmyadmin or anything .. insert "pages" tables. - to use .. access to http:///. 5. to add image. use phpmyadmin or anything to insert data into "images" tables. - to use image.. access to http:///i/. then it should be fine. ~
